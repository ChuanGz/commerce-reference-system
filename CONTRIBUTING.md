# Contributing

Commerce Reference System is a reference repository for learning and architecture discussion. It is not an open-ended community build.

## Before You Open Anything

- Read [LICENSE](LICENSE), [DISCLAIMER.md](DISCLAIMER.md), and [CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md).
- Use issues for factual defects, broken docs, and narrowly scoped technical corrections.
- Ask first before proposing broad feature work or structural changes.

## Accepted Input

- focused bug reports
- documentation corrections
- small fixes with a clear explanation
- constructive technical discussion that improves the repository

## Rejected Input

- copied code from elsewhere without clear rights
- large unsolicited feature work
- demands for free support
- disrespectful or low-value commentary
- attempts to reuse the repository as a commercial or internal starter without permission

## Submission Terms

By submitting code, text, diagrams, issues, or other material, you confirm that:

- you have the right to submit it
- it does not knowingly violate another party's rights
- you understand there is no obligation to review, merge, answer, or maintain it

You grant the repository owner a perpetual, worldwide, irrevocable, royalty-free, sublicensable right to use, modify, adapt, publish, redistribute, relicense, or reject the submission without compensation.

Do not contribute if you do not agree with these terms.
